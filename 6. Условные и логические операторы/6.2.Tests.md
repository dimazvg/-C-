# 6.2 if и else. Тесты

## **Тест 1. Что напечатает код?**

* Что напечатает эта часть кода (внутри main)  

```// листинг 1
int x = 8;
if (x > 4)
    printf("4");
if (x < 7)
    printf("7");
if (x == 8)
    printf("8");
```  

* Введите численный ответ
  * 48

## **Тест 2. Что напечатает код?**

* Что напечатает эта часть кода (внутри main)  

```// листинг 2
int x = 8;
if (x > 4) {
    printf("4");
    if (x > 9) {
        printf("9");
        if (x == 8) {
            printf("8");
        }
    }
}
```  

* Введите численный ответ
  * 4

## **Тест 3. Порядок имеет значение**

* Что напечатает эта часть кода (в main)?  

```// листинг 3
int x = 7;

if (x >= 8)
    printf("отлично");
else if (x >= 3)
    printf("удовлетворительно");
else if (x >= 5)
    printf("хорошо");
else
    printf("неудовлетворительно");
```  

* Выберите один вариант из списка
  * (-) ничего не напечатает
  * (-) отлично
  * (+) удовлетворительно
  * (-) хорошо
  * (-) неудовлетворительно
